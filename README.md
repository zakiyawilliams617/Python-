# Python 
This repository contains Python assignments organized by module. Each module focuses on a core programming concept, with a hands-on project to apply the material.

# Module 2 – Types, Operators, and Expressions
Assignment: Interest Rate Calculator
Explores Python's fundamental data types (integers, floats, strings, booleans) along with arithmetic, comparison, and logical operators. The interest rate assignment puts these building blocks to work by computing simple and/or compound interest based on user input of principal, rate, and time values.

# Module 3 – Control Structures and Conditional Statements
Assignment: Credit Card Checksum
Covers if/elif/else statements, for and while loops, and program flow control. The credit card checksum project applies the Luhn algorithm to validate credit card numbers, making heavy use of conditional logic and iterative processing.

# Module 4 – Data Structures
Assignment: N-Body Simulation
Introduces Python's core data structures — lists, tuples, dictionaries, and sets. The N-body simulation models the gravitational interactions between multiple celestial bodies, storing and updating position/velocity data across many time steps.

# Module 5 – Files, I/O, and Data Encoding
Assignment: K-Means Clustering
Focuses on reading from and writing to files, parsing data formats, and encoding/decoding data. The K-means clustering assignment reads in a dataset, applies the clustering algorithm iteratively, and writes the resulting cluster assignments back to a file.

# Module 6 – Functions
Assignment: Space Domain Awareness & Approximating Pi
Dives into defining and calling functions, scope, parameters, return values, and code reuse. Two projects are included: a space domain awareness tool (tracking or analyzing objects in orbit) and a Pi approximation program.

# Module 7 – REST APIs
Assignment: WMATA Accessibility
Covers how to make HTTP requests, handle JSON responses, and work with third party REST APIs. The WMATA (Washington Metropolitan Area Transit Authority) project queries the WMATA API to retrieve and display accessibility information, such as elevator/escalator status — across Metro stations.

# Module 8 – Object Oriented Programming 1
Assignment: Linear Feedback Shift Registers 
Built a pseudo random number generator using a custom LFSR class that performs bit shifting and XOR operations. Used it alongside the Pillow image library to encrypt and decrypt a PNG image by XORing each pixel's RGB values with LFSR generated values.

# Module 9 – Object Oriented Programming 2
Assignment: Generative Artificial Intelligence
Built a Retrieval Augmented Generation (RAG) pipeline in Python that fetches NASA's Astronomy Picture of the Day, extracts key terms using a local LLM (Llama 3.2 via Ollama), retrieves Wikipedia summaries for those terms, and generates an enriched image description by combining all sources.

# Module 10 – Developer Operations (DevOps)
Assignment: Building an Automated CI/CD Pipeline 
Created a Flask based REST API that wraps the WMATA Incidents API, allowing callers to filter results by unit type (elevator or escalator). Wrote unit tests using Python's unittest framework and optionally automated test execution on every Git commit using a CircleCI CI/CD pipeline.

# Module 11 – Data Science 1
Assignment: Hill Cipher
Implemented the Hill Cipher encryption algorithm in Python using NumPy, building a custom HillCipher class that encodes plaintext into numeric vectors, encrypts and decrypts using matrix multiplication and modular arithmetic, and handles noninvertible key matrices gracefully.
